puffins
=======

``puffins`` is a Python library for large-scale atmospheric and climate
dynamics research. It provides functions for computing physical quantities
and creating visualizations, primarily for use in Jupyter notebooks, with
built-in support for multi-planet calculations (Earth, Mars, Saturn, Titan,
and Venus).

.. note::

   This is a personal research tool with no official support. Functions
   assume specific input shapes and coordinate conventions (see
   :doc:`installation` and the API reference).
